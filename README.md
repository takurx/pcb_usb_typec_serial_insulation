# pcb_usb_typec_serial_insulation
This is pcb, usb type-c serial insulation

## image
This folder contains reference images and documentation photos for the PCB project. The images show the completed board design and layout. These visual references help users understand the physical appearance and configuration of the circuit board.

このフォルダには、PCBプロジェクトの参考画像とドキュメント写真が含まれています。画像は完成した基板設計とレイアウトを示しています。これらの視覚的な参考資料は、ユーザーが回路基板の物理的な外観と構成を理解するのに役立ちます。

## usb_typec_serial_insulation_adum1412
This folder contains the KiCad project files for a USB Type-C serial insulation board using the ADUM1412 digital isolator chip. The project includes schematic files, PCB layout files, and a complete set of Gerber manufacturing files in the gerber subdirectory. This design provides galvanic isolation for serial communication through USB Type-C connections.

このフォルダには、ADUM1412デジタルアイソレータチップを使用したUSB Type-Cシリアル絶縁基板のKiCadプロジェクトファイルが含まれています。プロジェクトには、回路図ファイル、PCBレイアウトファイル、およびgerberサブディレクトリ内の完全なガーバー製造ファイルセットが含まれています。この設計は、USB Type-C接続を介したシリアル通信のガルバニック絶縁を提供します。

## usb_typec_serial_insulation_adum121n
This folder contains the KiCad project files for a USB Type-C serial insulation board using the ADUM121N digital isolator chip. Similar to the ADUM1412 version, it includes complete schematic, PCB layout, and Gerber manufacturing files. This alternative design uses a different isolator IC to achieve serial communication isolation through USB Type-C.

このフォルダには、ADUM121Nデジタルアイソレータチップを使用したUSB Type-Cシリアル絶縁基板のKiCadプロジェクトファイルが含まれています。ADUM1412バージョンと同様に、完全な回路図、PCBレイアウト、およびガーバー製造ファイルが含まれています。この代替設計では、異なるアイソレータICを使用して、USB Type-Cを介したシリアル通信の絶縁を実現しています。
